Fox Food Hunt

  • Originality: It's a twist on the Phaser Tutorial game.
  • Technical Merit: It uses simple game physics to move the fox and also randomly generates food from a sprite sheet.
  • Prototype Postmortem: I mainly learned how to get my first game up and running, it is very similar to the First Game tutorial on Phaser but I learned how to use my own assets and generate random elements. The hardest part was getting the correct sizes to use for the sprites.
  • Prototype Assets:
    • Fox Sprite
    • Background
    • Food
    • Clouds
    • Ground platform
  • Prototype Closest Other Game: It closely resembles the Phaser Tutorial game.
    • I used code from the Tutorial Game: Making Your First Game
    • Also from code from here: Group vs Group
  • High Concept: Fox Food Hunt is a cute simple game where various foods are spread across the screen and you must collect, or rather eat them all.
  • Theme: There is not really a theme, it's a fox collecting different types of food.
  • Mandated Variety: For my input I used the arrow keys to move. For the randomness I used Math.random to generate a number between 0 and 500 to determine which food from the sprite sheet to use. This game is a platformer.
  • Prototype Goal: The mechanics include sprite movement/jumping and also item collectment.
  • Player Experience Goals: I want players to enjoy a cute and simple platformer game.
  • Gameplay: The player game move the fox to the left with the left arrow and to the right with the right arrow. The fox can jump using the up arrow, and can also jump on top of cloud platforms (but not through the platforms). By walking into the food the players can collect it for 5 points a piece. The goal is to collect all the food in the game.
  • Strategies: The players can use the cloud platforms to get food in places they can not reach by just simply jumping from the ground.
  • Story/Setting/Premise: The game is set place in a small hilly-type forest with a fox as the protagonist. The fox is very hungry, and has found a spot in the forest where there is a bunch of food! Thus the fox decides to try to eat them all.
  • Target Audience: This game is simple enough it can be played by anyone.
  • Play Time: 1 to 2 minutes.
